Bienvenue sur Arknights planner

Qu'est-ce que c'est ?

Résumons simplement le projet : Gamepress proposait un planner pour calculer les ressources nécessaires pour développer un operateur. Depuis un moment le site est down et le planner est devenu inutilisable. Du coup je suis partis pour faire le miens. Ce projet est à double objectif : recréer un planner facile d'accès et à comprendre et améliorer mes compétences de développement web en tant que développeur fullstack.
