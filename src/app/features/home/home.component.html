Bienvenu sur Arknights planner

Résumons simplement le projet : Gamepress proposait un planner pour calculer les ressources nécessaires pour développer un operateur. Depuis un moment le site est down et le planner est devenu inutilisable. Du coup je suis partis pour faire le miens. Ce projet est à double objectif : recréer un planner facile d'accès et à comprendre et améliorer mes compéténces de développement en web en tant que développeur fullstack.
